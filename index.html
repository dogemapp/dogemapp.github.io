DogEm App

Overview

Welcome to DogEm!


SP-4 DogEm is a project for CS 4850 - Fall 2022 - Section 03

DogEm is a cross-platform (iOS and Android) mobile app

that is used to persistently message your contacts

via call, text, and email until you get a reply.


DogEm is being developed with React Native using Expo CLI

The app's database is Firebase and it will be deployed to Expo Go

To use DogEm, users will install Expo Go then scan our QR code
